Описание игры "Ферма с поливом"

После запуска игры игрок попадает на главный экран с интерактивной фермой. Игра представляет собой симулятор фермы с механикой полива, где игрок должен выращивать различные растения с учетом их требований к расстоянию от воды.
Игра состоит из двух основных областей:
Панель инструментов - содержит все необходимые инструменты для работы с фермой:
-Курсор для просмотра информации о клетках
-Лопата для удаления растений
-Семена трех видов растений с разными требованиями к воде
-Ведро для создания и удаления водных источников

Игровая зона включает:
-Игровое поле 8x8 клеток с различными типами местности 
-Панель информации о выбранной клетке
-Подробные инструкции по игровому процессу

Игрок должен выращивать три вида растений, каждый из которых имеет особые требования к расстоянию от воды:
-Болотник требует воду на расстоянии 1 клетки
-Картошка требует воду на расстоянии 2-3 клеток
-Кактус требует воду на расстоянии 4+ клеток

Растения проходят через стадии роста (маленькие, средние, крупные) и могут погибнуть, если их требования к воде не выполняются. Игрок может изменять ландшафт, создавая и убирая водные источники с помощью ведра.

Особенности игры
Динамическая система роста растений с визуальными изменениями
Стратегический элемент размещения растений относительно водных источников
Возможность изменения игрового поля путем добавления/удаления воды
Информативная панель с деталями о каждой клетке
Адаптивный дизайн для различных размеров экранов
Игрок должен стратегически планировать расположение растений и водных источников для успешного выращивания всех видов культур. Визуально приятный интерфейс с анимациями и интуитивно понятным управлением делает игровой процесс увлекательным и доступным для игроков любого возраста.
